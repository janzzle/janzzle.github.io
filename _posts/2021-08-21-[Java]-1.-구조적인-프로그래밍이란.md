## 구조적 프로그래밍

: 절차로만 이루어진 프로그램을 나누어서 만드는 것
: 나누어진 코드 속에서 구조가 만들어짐

### 1. 구조적 프로그래밍의 방식

1) 기존의 절차로 이루어진 프로그램을 잘라서 구조를 만들어 가는 것 
→ Bottom up 방식

2) 코드가 없는 상태에서 구조부터 만들면서 코드를 만들어 가는 것
→ Top down 방식

프로그램 설계를 배운다면 Top-down 방식으로 시작해야하지만, 
코드를 나누는 방법부터 연습하기 위해 Bottom-Up으로 학습 시작
